# 🎨 Prixma Studio - Creative Portfolio System DEVFOLIO

Satu platform terintegrasi untuk desainer mengelola aset visual dan feedback klien dalam satu tempat. Build dengan **Next.js** dan **Tailwind CSS**.

### ✨ Kenapa Prixma Studio?
* **Real-time Collaboration:** Kirim revisi langsung di atas canvas design.
* **Automated Mockups:** Upload aset, dan biarkan sistem kami membuat mockups secara otomatis.
* **Developer Handover:** Export komponen UI langsung ke kode React.
