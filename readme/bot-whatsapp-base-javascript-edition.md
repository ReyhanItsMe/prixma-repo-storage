# 🤖 WhatsApp Bot Base (JavaScript Edition) `v0.2.0-beta`

Selamat datang di **WhatsApp Bot Base JavaScript Edition**. Ini adalah base bot yang dibangun menggunakan library **Baileys** yang sudah dioptimalkan agar ringan, responsif, dan mudah digunakan bagi para pengembang bot di Indonesia.

---

### ⚠️ Promo Khusus Beta
> [!IMPORTANT]
> Versi ini adalah versi **BETA** dan saat ini diberikan **100% GRATIS** (Harga normal Rp 50.000). Gunakan kesempatan ini untuk mencoba dan mengembangkan bot kamu sendiri! Harap maklum jika masih ditemukan bug.

### ✨ Fitur Unggulan
* 🚀 **High Performance:** Dibangun dengan core JavaScript yang ringan.
* 📲 **Dual Connection:** Mendukung koneksi via **QR Code** maupun **Pairing Code**.
* 🛠️ **Developer Friendly:** Struktur folder rapi dan mudah dimodifikasi.
* 📦 **Stable Base:** Menggunakan versi terbaru dari library Baileys.
